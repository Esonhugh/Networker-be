# networker-be

DN42 config generate Sever (Backend) 

一个用于自动收集 DN42 Peer 友友们配置的 Web 服务

带有自动导出朋友们配置数据的管理功能 

> 提供了一个 /admin/export.go 文件 
> 
> 编译后运行就可以导出所有的配置数据
> 
> 使用类似 [dn42ConfigGenerator](https://github.com/Esonhugh/dn42ConfigGenerator) 的工具
>
> 进行解析生成 就可以一键拿到所有的配置
